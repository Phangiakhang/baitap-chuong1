Chương số học

Bài 1: Kiểm tra số hoàn hảo

Số hoàn hảo là số có tổng các ước số nhỏ hơn nó bằng chính nó.

Input: Số nguyên dương N (1 ≤ N ≤ 109) Ví dụ:
Output: “Yes” nếu là số hoàn hảo, ngược lại xuất “No”
abc
abc
Input Output
371 Yes
324791234813534252 No

------------------------------------------------- Mã C++ ------------------------------------------------- 


#include 
using namespace std;

#define ll long long

bool soHoanHao(ll n)
{
	ll sum = 0;
	for (ll i = 1; i < n ; i++)
		if (n%i == 0) sum += i;
	return sum == n;

}

int main()
{
	#ifdef BITS_STDCPP_H 
		ios_base::sync_with_stdio(false);
		cin.tie(NULL); cout.tie(NULL);
	#endif 
	
	ll n;
	cin>>n;
	cout << (soHoanHao(n) ? "Yes":"No") << endl;
	return 0;
}
